# Claude Partner Network Learning Path

取得 Anthropic 合作夥伴認證的第一步

[anthropic.skilljar.com](https://anthropic.skilljar.com/page/claude-partner-network-learning-path)

---

## 什麼是 Claude Partner Network Learning Path？

這條學習路徑由 Anthropic 官方設計，作為加入 Claude Partner Network 的入門基礎，也是解鎖合作夥伴認證的第一個必要步驟。

完成這條路徑，你能得到三件事：

1. 系統性的技術知識框架 — 從 API、MCP 到 Agent，官方視角的完整脈絡
2. 合作夥伴認證資格 — 有出處、有說服力，是正式的 Anthropic 認可
3. 從「會用」到「能整合」的進化 — 每門課都直接對應實際的技術整合場景

---

## 四門課程總覽

| 課程 | 主題 |
|------|------|
| Introduction to Agent Skills | 建立、設定與共享可重複使用的 Agent 指令 |
| Building with the Claude API | 使用 Claude API 與 Anthropic 模型進行全面整合 |
| Introduction to Model Context Protocol | 建構 MCP 伺服器與客戶端，串接外部服務 |
| Claude Code in Action | 將 Claude Code 整合進開發工作流程 |

---

### 課程一：Introduction to Agent Skills

你會學到什麼

如何在 Claude Code 中建立、設定與共享可重複使用的 Markdown 指令（Agent Skills），讓這些指令在相關任務中自動套用。

為什麼重要

Agent Skills 是讓 Claude Code 從通用工具變成專屬助理的關鍵機制。學會設計與管理 Skills，能大幅提升重複性工作的執行效率。

---

### 課程二：Building with the Claude API

你會學到什麼

全面涵蓋如何使用 Claude API 與 Anthropic 模型進行開發整合——從基本呼叫到進階應用場景。

為什麼重要

API 是將 Claude 能力嵌入自有產品與工作流程的核心管道。這門課提供官方視角的完整技術框架，適合想深度整合 Claude 的開發者與技術人員。

---

### 課程三：Introduction to Model Context Protocol

你會學到什麼

使用 Python 建構 MCP 伺服器與客戶端，掌握三個核心功能：Tools、Resources、Prompts，讓 Claude 與外部服務直接串接。

為什麼重要

MCP 是 Claude 連接外部世界的標準協定。學會 MCP，就能讓 Claude 存取資料庫、呼叫 API、讀取本地資源，打開 Agentic 應用的真正邊界。

---

### 課程四：Claude Code in Action

你會學到什麼

如何將 Claude Code 深度整合進實際的開發工作流程，從個人使用到團隊協作的完整實踐。

為什麼重要

這門課把三部曲的實戰基礎與 Anthropic 官方的工程實踐對齊。帶著已有的使用體感來修，能把「為什麼這樣設計」的脈絡補完。

---

> 先取得 Anthropic 官方合作夥伴認證。
> 先實作、後修課。有了動手的體感，課程內容吸收效果更好。
> 你不只是「會用 Claude Code 的人」——你是通過官方認證的合作夥伴。
> 當你的整合成果開始被問「你怎麼做到的」，這條學習路徑讓你的方法有出處、有說服力。
