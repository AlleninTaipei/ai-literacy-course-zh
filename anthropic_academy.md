# Anthropic Academy

Anthropic 官方免費學習平台

[anthropic.skilljar.com](https://anthropic.skilljar.com)

---

## 三個你需要知道的事實

1. 完全免費——只需電子郵件，無需訂閱或信用卡
2. 官方出品——內容由 Anthropic 原廠製作，不是第三方詮釋
3. 完課即可下載 Anthropic 官方證照——直接加進 LinkedIn，讓專業檔案說話

> 這不是教你「怎麼用 ChatGPT」的業配課。這是做出 Claude 的那支團隊，親自告訴你他們的模型是如何工作的。

---

## 五大學習軌道

| 軌道 | 定位 | 適合誰 |
|------|------|--------|
| AI Fluency | AI 素養入門 | 所有人 |
| Product Training | Claude 日常應用 | 白領工作者 |
| Developer Deep-Dives | API 與架構實作 | 開發者 |
| Cloud & Enterprise | 企業規模導入 | IT 決策者 |
| Foundational Knowledge | 基礎知識補課 | 任何需要補底的人 |

截至 2026 年 5 月，平台已開放 17 門課程，持續增加中。

---

## AI Fluency 軌道

核心課程：AI Fluency: Framework & Foundations

你會建立的框架——**4E**：

- **Effective**：讓 AI 真正幫你完成工作
- **Efficient**：不浪費，知道何時用、何時不用
- **Ethical**：理解 AI 的邊界與責任
- **Safe**：辨識風險，做有依據的決策

---

## Product Training 軌道

核心課程：Claude 101 / AI Capabilities and Limitations

你會學到：

- 如何用 Claude 處理日常工作——寫郵件、整理資料、分析文件、草稿內容
- 理解 Claude 能做什麼、不能做什麼，避免誤用
- 從「試試看」進化到「穩定納入工作流程」

---

## Developer Deep-Dives 軌道

七門課涵蓋完整的 Claude 開發技術棧：

| 課程 | 主題 |
|------|------|
| Building with the Claude API | API 呼叫到進階整合 |
| Introduction to Model Context Protocol | MCP 伺服器與客戶端實作 |
| Claude Code 101 | Claude Code 核心使用方式 |
| Claude Code in Action | 整合進開發工作流程 |
| Introduction to Subagents | 多代理架構入門 |
| Introduction to Agent Skills | 建立可重複使用的 Skills |
| Introduction to Claude Cowork | 團隊協作場景實作 |

## 完課即得：Anthropic 官方證照

每門課完成後，平台自動頒發電子證照，你可以：

- 一鍵加進 **LinkedIn 個人檔案**，在徵才市場中被看見
- 附在履歷或作品集，作為 AI 能力的具體佐證
- 向主管或客戶說明你的 AI 知識有官方出處，不是自學摸索

證照由 Anthropic 直接背書，不是第三方平台的結業證明。

1. 前往 [anthropic.skilljar.com](https://anthropic.skilljar.com)

4. 完課後下載官方證書

全程免費，隨時可開始。
