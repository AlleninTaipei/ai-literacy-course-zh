# Anthropic Academy 官方課程學習指南

從入門到認證，系統性累積你的 AI 實力

[anthropic.skilljar.com](https://anthropic.skilljar.com)

---

## 為什麼要修 Anthropic Academy？

Anthropic Academy 是 Claude / Claude Code 的官方學習平台。

修習這些課程，你能得到三件事：

1. 系統性的 AI 知識框架 — 補足實作中容易忽略的觀念基礎
2. 官方認證證書 — 有出處、有說服力，不只是「我有在用 AI」
3. 從「會用」到「擅用」的進化 — 每門課都和實際應用場景高度銜接

---

## 建議學習路徑

| 優先順序 | 課程 | 理由 |
|---------|------|------|
| ★ 優先修 | Claude Code 101 | 三部曲的官方補完，補齊設計邏輯 |
| ★ 優先修 | Introduction to Claude Cowork | 三部曲的延伸，協作品質推向下一層 |
| 選修 | AI Capabilities and Limitations | 角度不同，有實作經驗後再修效益最大 |

> 三部曲已完整涵蓋 AI 基礎概念與商業應用場景，不需要重複修習同類課程。把學習時間集中在真正能帶來增量的地方。

---

### 課程一：Claude Code 101 ★ 優先修

你會學到什麼

Anthropic 官方設計的 Claude Code 入門課，涵蓋核心概念、基本操作與最佳實踐。

為什麼現在修最合適

你已經完成了安裝與初步使用，帶著實作體感來修這門課，能把「為什麼這樣設計」的脈絡補完——不會只是背操作步驟，而是真正理解官方的設計邏輯。

---

### 課程二：Introduction to Claude Cowork ★ 優先修

你會學到什麼

與 Claude 協作的進階實作訓練——從「偶爾問問 AI」到「讓 AI 成為真正的工作夥伴」。

為什麼現在修最合適

三部曲的終點，正是這門課的起點。你已具備框架與工具，這門課把協作品質推到下一層。這門課的設計前提是你已經有足夠的使用量——帶著實際工作的疑問來修，才能真正吸收。

---

### 課程三：AI Capabilities and Limitations（選修）

你會學到什麼

系統性了解 AI 真正能做什麼、不能做什麼，以及能力邊界在哪裡。

為什麼是選修

三部曲已從「它是什麼」的角度談了幻覺、知識截止日、對齊稅；這門課從「能/不能」的系統性地圖切入，角度不同但不衝突。實際踩過 Claude Code 的邊界之後再修，收穫比課前大。有興趣深化能力邊界認知的學員再修即可。

---

> 取得 Anthropic 官方證書。
> 先實作、後修課。有了動手的體感，課程內容吸收效果更好。
> 你不只是「會用 Claude Code 的人」——你是拿到官方認證的那個。
> 當你的工作成果開始被問「你怎麼做到這麼快」，這些證書讓你的方法有出處、有說服力。
